# LightSignal Market Intel

Market intelligence pipeline for Lightpath — tracks data center and network
infrastructure investment signals across the US.

## Quick Start

### Article pipeline (run twice daily)
```
python scripts/articles/run_articles.py
```

### Full dashboard build (run once daily, after article pipeline)
```
python scripts/run_all.py
```

### Manual modes
```
python scripts/articles/run_articles.py --fetch-only     # RSS only
python scripts/articles/run_articles.py --skip-fetch     # skip RSS, process staged
python scripts/articles/run_articles.py --classify-only  # classify only
python scripts/run_all.py --skip-h3                      # skip H3 regen (faster)
```

## Environment Variables

